# Liel Pets

A modern, responsive pet store and adoption website designed with a clean aesthetic and user-friendly interface. This project was developed as part of my first-year web development studies.

## Features

- **Interactive UI**: A sleek turquoise-themed design with a focus on visual storytelling.
- **Animated Galleries**: Smooth CSS animations for pet showcases, including jumping parrots and fading background transitions.
- **Smart Navigation**: A custom dropdown menu for easy browsing of categories like "Our Animals" and "Our Products".
- **Adoption Center**: A streamlined adoption form for potential pet owners.
- **Responsive Layout**: Built using Flexbox and Grid to ensure a seamless experience across different screen sizes.

## Technologies Used

- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3**: Custom styling, Flexbox/Grid layouts, and Keyframe animations.
- **Git & GitHub**: Version control and deployment via GitHub Pages.

## Preview

The website features high-quality visuals of parrots, hamsters, and rabbits, creating an engaging experience for animal lovers.
